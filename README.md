# E-Commerce-Website-Using-Python

## Platform and technologies Used
### Front-End
 - HTML5<br>
- CSS3<br>
- JavaScript<br>  
- Bootstrap<br>  

### Back-End
  - Python - Flask <br>
  - MySQL <br>

## Key Features
### Public User
- Search Products <br>
- View Product details<br>
- Create a User Account <br>

### Signed-in User
- Search and view Products <br>
- Create an Order <br>
- Update email and password<br>
- View previous orders with **Update** and **Delete** options  
 <br>

### Admin Panel
- Add New Products <br>
- Update existing Products <br>
- view all Orders <br>
- Manage all registered Users <br>
